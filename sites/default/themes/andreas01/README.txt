This theme is a port for Drupal of the andreas01-theme as found on 
http://openwebdesign.org/viewdesign.phtml?id=2199

This theme requires PHPTemplate.
It validates as XHTML1.1 and CSS2

Thanks to Tobias Maier for pointing out some issues and providing fixes.

If you are using the Image-module, be sure to set the size of previews to 448 or lower.

The logs in the Admin-section are given a very small font in order to fit in the space available.
